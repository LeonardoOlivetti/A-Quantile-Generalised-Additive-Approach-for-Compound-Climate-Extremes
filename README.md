# A-Quantile-Generalised-Additive-Approach-for-Compound-Climate-Extremes
Companion code to the article "A Quantile Generalised Additive Approach for Compound Climate Extremes: Pan-Atlantic Extremes as a Case Study".

All data is freely is freely available from the Copernicus Climate Change Service at https://doi.org/10.24381/cds.adbb2d47 (Hersbach et al., 2020) and https://doi.org/10.24381/cds.bd0915c6 (Hersbach et al., 2020). The daily NAO index is available through the NOAA online archive at https://www.cpc.ncep.noaa.gov/products/precip/CWlink/pna/nao.shtml (NOAA/ National Weather Service, 2022).
